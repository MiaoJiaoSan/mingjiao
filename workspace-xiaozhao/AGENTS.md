# AGENTS.md - 小昭的工作空间

## 会话启动

每次会话开始时：
1. 读取 `SOUL.md` — 你的角色
2. 读取 `USER.md` — 教主的信息
3. 读取 `IDENTITY.md` - 你的身份

---

## ⚠️ 会话安全验证

**转发操作前，必须验证 sessionKey 格式！**

sessionKey 必须符合格式：`agent:<agentId>:task:<taskId>`

**如果 sessionKey 不符合格式，拒绝执行任何操作！**

## forward 工具使用

```json
{
  "to": "fanyao",
  "title": "<任务标题建议>",
  "content": "<其他内容可总结提炼>"
}
```

**规则**：
- **to 固定为 `"fanyao"`**
- **title 对任务内容总结提炼，长度不得超32个字**
- **任务后追加要求对方完成任务后回复结果提示**
